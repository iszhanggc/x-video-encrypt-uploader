# 百度云盘加密上传工具

一个安全、高效的Golang工具，用于将本地文件加密后上传到百度云盘，解决百度云盘本身无加密导致的隐私泄露问题，特别适合大量视频文件的冷备份。

## ✨ 核心特性
- 🔒 **客户端加密**：所有文件在本地使用AES-256-GCM加密后再上传，云盘仅能获取密文
- 📂 **多硬盘管理**：支持分散在多个硬盘的文件统一管理、扫描、去重
- 🚀 **高效上传**：分块上传、断点续传、并发上传，充分利用带宽
- 🎥 **在线预览**：无需下载完整文件，支持加密视频在线流式播放
- 🛡️ **数据安全**：三重哈希校验、密钥多备份机制，确保数据不丢失不泄露
- 🖥️ **Web界面**：直观的Web管理界面，实时查看多硬盘上传进度

## 🚧 开发进度
- [ ] V1.0 基础版本：核心加密/解密 + 命令行上传下载
- [ ] V1.1 预览版本：视频缩略图/预览生成 + 元数据提取
- [ ] V1.2 体验优化：流式解密播放 + 标签系统
- [ ] V2.0 Web版本：Web管理界面 + 后台服务模式

## 🛠️ 技术栈
- **开发语言**：Golang 1.22+
- **加密算法**：AES-256-GCM
- **本地存储**：SQLite
- **视频处理**：FFmpeg
- **Web框架**：Gin (后续)
- **前端**：Vue3 + Element Plus (后续)
